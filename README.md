# social-media-app
Facebook clone

## Tech stack
* Frontend: ReactJS
* Database: MongoDB
* Infrastructure: Docker
* Backend: ExpressJS, Redis

## Run project in local
* Create file .env in fronend folder
* Create file .env in backend folder
* Run command ```docker compose up```
